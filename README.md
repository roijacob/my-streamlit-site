## Running this app

1. Create a virtual environment in this repository:
   ```
   python -m venv .venv
   ```

2. Activate the virtual environment:
   - For Windows:
     ```
     .venv\Scripts\activate
     ```
   - For macOS and Linux:
     ```
     source .venv/bin/activate
     ```

3. Run the app using Streamlit:
   ```
   streamlit run app.py
   ```
